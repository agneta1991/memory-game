ARE YOU READY TO PLAY A GAME?

INSTRUCTIONS

Memory card game is a fun and exciting game that can be played by people of all ages. In this game, you need to flip over cards in an attempt to match pairs. However, there is a twist that you need to be aware of. If you flip a card and a joker is found, you lose a life. You have just 2 lives, so try to remember where the joker is. if card of a Joker is pressed twice, it is a game over.

Opened cards will be visable just 3 seconds, so don't rush but try memorising it as fast as you can. This adds an extra layer of excitement to the game and encourages you to be strategic in your moves.

GOOD LUCK!
